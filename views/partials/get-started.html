1. Install node.js

Download here https://nodejs.org/en/download/

2. Play around with command line

- On window type cmd on run/search box

- Type node on command line, nodejs will runup

- Type var a = 1 on command line

- Type a on command line, 1 will show up

- Type var b = 2 on command line

- Type a + b on command line, you will get 3

- As you can see, this is look like javascript console but it's run on PC or we can tell it's run on server side

The things that we should know here:

- We can type global to see global variable

- We can type node -v to see the nodejs version, but if it's in running mode type process.version instead

- We can type ctrl + C to stop nodejs from running

- Alert command is not working don't worse your time to try it!

- But console.log() still work, YEAHH !

3. Run basic js file

- Use command line go to the folder that contain .js file

- Type node [filename].js (without .js still work)

- Let's try on this project, go to folder example

- Type node hello.js then see the result

- Then type node server.js open browser and then go to 127.0.0.1:9000 to see the result
