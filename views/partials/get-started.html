4. Nodejs module, imports(require), exports

- Understand nodejs module

- Imagine that this line of code exists as the first line of code exports = module.exports = {}

- Imagine that require is function (path) { return module.exports; }

- Let's create file calc.js and put this code in:

exports.plus = function (a, b) {

return a + b;

}

exports.minus = function (a, b) {

return a - b;

}

module.exports.multiply = function (a, b) {

return a * b;

}

module.exports.divide = function (a, b) {

return doDivide(a, b);

}

function doDivide(a, b) {

return a / b;

}

- Then let's create file math.js and put this code in:

var math = require("./calc.js");

console.log(math.plus(1, 2));

console.log(math.plus(2, 1));

console.log(math.plus(2, 2));

console.log(math.devide(10, 2));

- Try to execute math.js file to see the result type: node math on command line tool

5. NPM

- What's npm? npm is the package manager for javascript platform.

- if you are .NET developer then, imagine it's nuget for javascript platform, OK?.

- Here is the link to learn more about NPM: https://docs.npmjs.com/

The command that we should know here:

create package.json (initialize): type npm init

install package: npm install [packagename]

Let's try npm install jquery

install package with version: npm install [packagename]@[version]

Let's try npm install jquery@1.11.1

Save package to packages.json npm install jquery@1.11.1 --save-[flag](optional)

Update npm npm install npm -g
