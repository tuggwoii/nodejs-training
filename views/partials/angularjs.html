1. What's AngularJS and Why AngularJS?

- Let's start with ASP.NET MVC, the main logic cod is C# and the view render is ASPX or Razor

- What's about php, the the main logic code is php and view render is still php

- Node.js main logic code is JavaScript and view render is JavaScript

It's look normal so what's the problem?

- If you try to use JavaScript to creat your own framework you will get a hard time to doing that

- Same as if you try to render the view with javascript

- So we need a good frame work here

- For me the solution will be AngularJs, because it's have 2 things that I want in 1 js library

- But the combination with mutiple js library still OK for node.js

So what's AngularJS

- MVC JavaScript framework

- Dynamic HTML, HTML render/compiler

- Data binding

4. Setup AngularJS

Easy like previous chapter npm install angular --save

Start with ng-app one of the AngularJS directive

1. add ng-app to html element that we want to use AngularJS script

2. Remember that only 1 ng-app is allowed in 1 pages

3. So if you want to use in entire page the best choice is add it on body element

4. Added it as an attribute like this <body ng-app="appName">

5. After add you will get error message like this Uncaught Error: [$injector:modulerr] http://errors.angularjs.org/1.4.7/$injector/modulerr?p0=app&p1=Error%3A%20%â€¦2F%2F127.0.0.1%3A5000%2Fnode_modules%2Fangular%2Fangular.min.js%3A19%3A463)

6. Create .js file to connect with angular module that we attach on html element. I wil create file and name it as app.js

7. Place this code in the file var app = angular.module('appName', [ ]); then reference the script file in to the html file under the AngularJs script

8. Error message must disappear and we'll done on setup AngularSs

5.4 Useful View render/compile Directives

- ng-include load html file to append under the biding element, attribute value must be url of the file

- ng-show, ng-hide show or hide element, attribute value must be boolean

- ng-if remove or create element, attribute value must be boolean

- ng-repeat repeat create element, attribute value must be array

5.5 Form Directives

- ng-model use for recieve value from input box, attribute value must be boject that we want to assign value to

5.6 Custom Directive

- Directive is a magic that transform simple html into functional html

- Directive is detect by 3 type call as restrict

- Restrict: A = attribute, E = element, C = class

- We can combination restrict like AE EC AEC if one of them are meet then it will work

- If no restrict it will set to AE

- If the html match the restrict it will transfrom to what we write on directive

- learn more here https://docs.angularjs.org/guide/directive
