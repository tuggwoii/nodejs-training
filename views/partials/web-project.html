1. Let's create web project

- We going to start with create folder on your drive.

- Go to that folder with command line tool

- Start with init npm packages by type npm init

- Insatll express npm install express --save

- Create 4 folder call images, scripts, css and views mkdir [folder name]

- Create file name index.js echo. 2>index.js

- Create file name index.html in views/pages folder echo. 2>index.html (first you need to navigate to views/pages directory)

- Open your root folder as a website on visual studio

- Open index.js and write code like this: index.js into the file

- Open index.html and write code like this: index.txt into the file

- Final step, run the index.js file node index
